# Hackathon-Finance-Agents: Circle Arc

Arc is a purpose-built Layer 1 (L1) blockchain by **Circle** designed as the "Economic OS" for stablecoin finance. This repository focuses on building financial agents leveraging Arc's developer-first infrastructure.

## 🛠 Developer Core Primitives

### 1. Gas Mechanics (Native USDC)
Arc eliminates the friction of holding volatile native tokens for network fees.
- **Native Gas:** USDC is the native token used to pay for transaction fees.
- **Impact:** Simplifies user onboarding and corporate accounting by removing the need to manage secondary assets (like ETH or SOL).

### 2. Performance & Finality
The network utilizes the **Malachite** consensus engine to provide institutional-grade speed.
- **Sub-second Finality:** Transactions reach deterministic finality in < 1 second.
- **Impact:** Eliminates the need for "confirmation counting." Once a transaction is processed, it is permanent and immutable.

### 3. Integrated Infrastructure
- **Built-in FX:** A protocol-level engine for swapping between USDC, EURC, and other local stablecoins.
- **Opt-in Privacy:** Selective, compliant privacy layers designed to meet institutional confidentiality requirements.

## 📅 Roadmap

- **Mainnet Beta:** Expected 2026.
